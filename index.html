HTML

  • HTML stands for hypertext markup language
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Can create paragraphs, headers etc

CSS

  • CSS stands for Cascading Style Sheets, it is a programming language used to show how content will be presented to users of the webpage
  • CSS defines the style, colour, font etc.
  • 3 ways to style, inline CSS, internal CSS and external CSS (external CSS are preferred because it makes design changes much easiers)
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • Git allows you to keep track of versions of projects
  • Git allows you to track your own progress on a project without interfering with a colleagues work on the same project
  • Git allows you to work on feature braches so that you do not interfere with the main branch

JavaScript

  • Allows users to interact creatively within a website
  • To declare a variable we use "var"
  • here we can create functions, the "fundamental building blocks"
  • control flow is the order the computer computes the tasks- must be done from top to bottom
